Assignment 3

Dataset Description

This dataset contains video game sales. Each row provides details of a game with attributes such as 'name', 'platform', 'year', 'genre', and 'publisher'. For every game there is also numeric sales fields (in millions of units) alongside details of 'sales region' and amount.

Variables & Data Types

  • Name — categorical (nominal)
  • Platform — categorical (nominal)
  • Year — ordinal/quantitative
  • Genre — categorical (nominal)
  • Publisher — categorical (nominal)
  • NA_Sales, EU_Sales, JP_Sales, Other_Sales, Global_Sales — quantitative
